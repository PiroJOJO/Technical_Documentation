Описание

Катаракта является одним из самых распространенных заболеванием глаз в мире, которое характерно для пожилого населения. Причиной помутнения хрусталика является не только возраст, но и множество других факторов, которые связаны с эндокринной системой человека, генетической предрасположенностью и состоянием экологической обстановки, что является большой проблемой на сегодняшний день. С катарактой сталкиваются 8,4% людей трудоспособного возраста в России. Помутнение хрусталика ограничивает человека в получении визуальной информации, приводит к истощению жизнедеятельности и к полной потери зрения, процентное значение которой равен 22,2% для молодого населения нашей страны. Вследствие этого, ранее квалифицированное офтальмологическое обследование дает возможность подобрать корректное хирургическое вмешательство для восстановления функции глаз.

Во время диагностики глаз пациента подвергают к различным методам обследования. Проверяется общий анализ крови, острота зрения, чувствительность к свету. Проводится метод исследования с помощью щелевой лампы, чтобы получить биомикроскопические изображения хрусталика для оценки его плотности ядра. При данном анализе используется классификация помутнений хрусталика, разработанной в 1993 году, LOCS-Ⅲ для оценки вида и степени катаракты. Такая классификация представляет собой таблицу, где для каждого примера вида и степени заболевания прилагается эталон-изображение. Таким образом, специалисту необходимо визуально определить, оценить схожесть данных, полученных с помощью биомикроскопии, с шаблонами из системы классификации LOCS-Ⅲ.

Развитие технологического прогресса дает возможность решать многообразные задачи связанные с высокозатратными вычислениями в разных сферах деятельности человека. Нейронные сети приспосабливают под различные проблемы для поиска более сложных закономерностей, которые улучшают точность прогнозирования и решения. По этой причине, предсказание вида катаракты с помощью искусственного интеллекта будет не только эффективно, но и практически полезно, как инструмент для офтальмологического обследования и микрохирургии глаза.

На сегодняшний день отсутствуют программы, которые способны решать проблему классификации катаракт по изображениям, полученных с помощью щелевой лампы, что дает возможность быть открытой сферой в биомедицине для исследования и разработки в рамках машинного обучения и искусственного интеллекта. Поэтому целью данной работы является реализация алгоритма на основе нейронных сетей для распознавания видов катаракт по системе классификации помутнений хрусталика LOCS-Ⅲ. Для достижения результата выполнялись следующие задачи:

  • Сформировать собственный датасет на основе биомикроскопических изображений глаза для классификации хрусталика.
  • Реализовать алгоритм распознования тип и степень катаракты глаза по системе LOCS-Ⅲ.
  • Используя алгоритм, охарактеризовать изображения глаз по системе классификации помутнений хрусталика LOCS-Ⅲ.
